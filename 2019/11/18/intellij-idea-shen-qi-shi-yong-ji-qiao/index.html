第七章：开始调试你的程序

![1551797888819](IntelliJ IDEA神器使用技巧.assets\1551797888819.png)

resume：直接跳到下一个断点或者结束程序运行

查看所有断点：shift + ctrl +f8

禁止所有断点：Mute Breakpoints,让所有断点都变成白灰色，按下F9跳过所有断点执行

条件断点：当要执行某个断点的时候，必须要满足某个条件这个断点才会生效，ctrl+shift+f8，然后输入需要满足的条件，条件是合法的java表达式。

条件断点的符号是：红色断点旁边有一个问号

表达式求值：evaluate expression,用得比较多；

运行到制定行：Run to Cursor，运行到光标的指定行

setValue:按下F2改变变量的值；

Run anywhere if you can

![1551799374729](IntelliJ IDEA神器使用技巧.assets\1551799374729.png)

debug configuration

光标停留的地方按住ctrl+shift+f9就可将该段程序运行起来，找运行的上下文，然后找到最小的上下文；

可以从历史列表中选择一个运行，run–>debug

edit configuration:编辑当前运行，可以修改运行的函数参数等
